# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Professional GitHub repository setup

- Code quality checks

## [3.0.0] - 2025-11-18

### Added
- Full project reorganization into modular structure
- NIFTY500 processing tools and data
- Progress reports and tracking
- Comprehensive documentation

### Changed
- Directory structure reorganized into numbered modules
- Improved documentation structure

## [2.0.0] - 2025

### Added
- Generic Stock Analyzer toolkit (universal CSV support)
- Pattern detection (35+ patterns)
- Statistical analysis (26+ technical indicators)
- Fundamental metrics analyzer
- Comprehensive visualization generator
- Automated report generation

### Changed
- Refactored analysis pipeline for any stock

## [1.5.0] - 2025

### Added
- Complete Reliance Industries analysis example
- 6,433 trading days analyzed (2000-2025)
- Verification and audit trails
- Pattern visualizations

## [1.0.0] - 2025

### Added
- Initial release
- Core fundamental scoring system
- 14-metric calculator
- Weighted scoring engine (0-100 scale)
- Excel report generator
- Eternal Ltd sample analysis
